Actions

ตั้งกฎ: เมื่อเกิดเหตุการณ์ (ได้ของขวัญ / แชทคำสั่ง / ครบไลค์ / กดคีย์ลัด / สุ่มรางวัลออก ฯลฯ) → ให้ทำอะไร (แจ้งเตือน, อ่านเสียง, เล่นเสียง, กดปุ่มในเกม, สั่ง OBS, ยิง webhook, สุ่มรางวัล)

เทมเพลตยอดฮิต — กดเพื่อเพิ่มแล้วปรับต่อได้

Widgets สำหรับ OBS

วิธีเพิ่มใน OBS: Sources → + → Browser → วาง URL ด้านล่าง → กำหนดขนาดตามที่แนะนำ → พื้นหลังโปร่งใสอัตโนมัติ (แนะนำติ๊ก "Shutdown source when not visible" และ "Refresh browser when scene becomes active")

เสียงอ่าน (TTS)

อยากได้เสียงไทยเพิ่ม/คมชัดขึ้น (โหลดเสร็จกดปุ่มทดสอบด้านล่างได้เลย):
macOS: System Settings → Accessibility → Spoken Content → System Voices → จัดการเสียง → โหลดเสียงไทย (Kanya พรีเมียม / Siri)
Windows: Settings → Time & Language → Speech → Manage voices → Add voices → เลือกภาษาไทย (Microsoft Pattara)

สุ่มรางวัล (Roulette)

ตั้งรายการรางวัลด้านล่าง → เพิ่ม widget "Roulette สุ่มรางวัล" ใน OBS (ดูแท็บ Widgets) → สั่งสุ่มได้ 3 ทาง: สร้าง Action ที่มีการกระทำ "สุ่มรางวัล" (เช่น เมื่อได้ของขวัญที่กำหนด), ตั้งคีย์ลัด, หรือกดปุ่มสุ่มทดสอบในหน้านี้
