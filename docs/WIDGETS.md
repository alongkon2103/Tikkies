# 🖥️ Widgets สำหรับ OBS — เอกสารอ้างอิง

ทุก widget เสิร์ฟจาก overlay server ของ Tikkies Tools ที่ `http://localhost:<พอร์ต>/widgets/<ชื่อ>.html`
(พอร์ตเริ่มต้น **21213**) พื้นหลังโปร่งใสอัตโนมัติ เหมาะกับใช้เป็น **Browser Source** ใน OBS

## วิธีเพิ่มใน OBS
1. **Sources → + → Browser**
2. วาง URL ของ widget (คัดลอกจากแท็บ **Widgets** ในแอป)
3. ตั้ง **Width/Height** ตามขนาดแนะนำ
4. แนะนำติ๊ก **"Shutdown source when not visible"** และ **"Refresh browser when scene becomes active"**
5. ปรับแต่งเพิ่มด้วย **Custom CSS** ของ OBS ได้ (เช่น `body { transform: scale(1.2); }`)

พารามิเตอร์ทั่วไปทุก widget: `?host=` และ `?port=` (กรณีเปิดจากเครื่อง/พอร์ตอื่น)

## 🔔 alerts.html — Alert Box
แจ้งเตือนกลางจอเมื่อได้ของขวัญ/ติดตาม/สมาชิก/แชร์ และแสดง Alert จากระบบ Actions
**ขนาดแนะนำ:** 500 × 300

| พารามิเตอร์ | ค่าเริ่มต้น | ความหมาย |
|---|---|---|
| `gifts` | `1` | แสดง alert ของขวัญ |
| `follows` | `1` | แสดง alert ผู้ติดตามใหม่ |
| `subs` | `1` | แสดง alert สมาชิกใหม่ |
| `shares` | `0` | แสดง alert การแชร์ |
| `mindiamonds` | ตาม settings | เพชรขั้นต่ำที่จะแสดง alert ของขวัญ |
| `duration` | ตาม settings (6) | ระยะเวลาแสดงต่อ alert (วินาที) |
| `sound` | `1` | เปิด/ปิดเสียง (`0` = ปิด) |
| `debug` | `0` | แสดงปุ่มทดสอบลอยบนจอ |

## 🔊 tts.html — TTS Caption
แสดงคำบรรยายข้อความที่กำลังอ่าน และ (ถ้าเปิดในเบราว์เซอร์จริง) อ่านออกเสียงด้วย
**ขนาดแนะนำ:** 600 × 120

| พารามิเตอร์ | ค่าเริ่มต้น | ความหมาย |
|---|---|---|
| `caption` | `1` | แสดงคำบรรยาย (`0` = ปิด) |

> ⚠️ OBS Browser Source ส่วนใหญ่**ไม่รองรับ** `speechSynthesis` — ให้เปิดโหมด **"อ่านในแอป" (playInApp)**
> ในแท็บ TTS เพื่อให้ตัวแอปเป็นคนอ่านเสียง แล้วใช้ widget นี้แสดงคำบรรยายอย่างเดียว

## 🔧 เทคนิค OBS

- **ปรับขนาด/ตำแหน่ง:** ลากขอบ source ใน OBS ได้ตามปกติ; widget ออกแบบให้ยืดหยุ่น
- **โปร่งใส:** ทุก widget ตั้ง `background: transparent` แล้ว — ไม่ต้องตั้งค่าเพิ่ม
- **หลาย widget:** เพิ่มได้หลาย Browser Source พร้อมกัน (แต่ละอันเชื่อม WebSocket แยก)
- **รีเฟรช:** ถ้า widget ค้าง กด **Refresh** ที่ source (widget จะ reconnect WebSocket อัตโนมัติอยู่แล้ว)
- **Custom CSS:** ใส่ใน properties ของ Browser Source เพื่อ override สไตล์ เช่นเปลี่ยนฟอนต์/ขนาด
